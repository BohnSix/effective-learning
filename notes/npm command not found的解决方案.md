２、依次输入下列命令
```
        sudo apt-get remove npm

        sudo apt-get remove nodejs-legacy

        sudo apt-get remove nodejs 

        sudo rm /usr/bin/node 

        sudo apt-get install nodejs 

        sudo apt-get install nodejs-legacy 

        sudo apt-get install npm.
```
